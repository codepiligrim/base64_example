Encode

- Put any file into "src" folder
- Run script: b64.py encode %YOUR_FILE_NAME%
- Grab encoded string in "encoded" folder %YOUR_FILE_NAME%.b64

Decode

- Put your base64 string file to "src_b64" folder
- Run script:  b64.py encode %YOUR_FILE_NAME%.b64
- Grab decoded file in "decoded" folder %YOUR_FILE_NAME%
